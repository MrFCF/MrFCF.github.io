    npm install –global gulp-cli
  2. 在项目中devDependencies中安装gulp


    npm install –save-dev gulp
  3. gulpfile.js在项目的根部创建一个：


  4. 运行gulp

    gulp

    要运行单个任务，请使用gulp< task >< othertask >.

gulp插件的使用

  1. 首先先引入你要使用的插件

    gulp.src 用来获取你写的scss文件的地址

    s.pipe(fn)方法传入方法的是一个function，这个function作用无非是接受上一个流（stream）的结果，并返回一个处理后流的结果(返回值应该是一个stream对象)。
    options.bundleExec(布尔值，默认值：false)


    当这个选项为真时，bundle exec scss用来代替scss。否则， scss用来代替bundle exec scss


    gulp.dest(为当编辑完成后的css文件的输出位置)

运行gulp代码和监听watch

运行gulp

  1. gulp.watch第一个参数 为要监视的文件匹配模式 ./templates/*.hbs为所有templates文件目录下的以.hbs结尾的文件
  2. 第二个参数是一个数组为一个可选的配置对象
  3. 第三个参数是一个回掉函数，每当监视文件发生变化时，就会调用这个函数，并且会给他传入一个对象，该对像包含了文件发生变化的一些信息，type属性为变数的类型 例如：

    gulp.watch(‘js/*/.js’,function(event){
    console.log(event.type);变化类型
    })
